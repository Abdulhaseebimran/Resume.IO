Resume.IO Pro is a world-class, high-fidelity AI-powered resume builder designed to help professionals land their dream jobs. It leverages advanced AI to generate high-impact content, optimize for ATS (Applicant Tracking Systems), and provide real-time feedback through a stunning, modern interface.

---

## ✨ Key Features

- **🧠 Intelligent Content Generation**: Powered by advanced AI models (Claude/Gemini) to craft professional bullet points and summaries.
- **📊 Real-time ATS Scoring**: Get instant feedback on how well your resume is optimized for recruiter systems.
- **🎨 Premium Templates**: A curated gallery of world-class, responsive templates (Minimalist, Executive, Creative, and more).
- **🔒 Secure Authentication**: Robust Login/Signup system with Google OAuth and standard Credentials support.
- **📱 Responsive Excellence**: A "wow" experience across all devices, from elite desktop mockups to mobile-first editing.
- **📈 View Analytics**: Track how many times your shared resume has been viewed.
- **🌗 Elite Dark/Light Mode**: Full theme support with premium glassmorphic components.

---

## 🛠️ Technology Stack

- **Frontend**: Next.js 16 (App Router), React 19, Tailwind CSS 4.
- **Backend**: Next.js API Routes, Next-Auth (v4).
- **Database**: MongoDB with Mongoose.
- **AI Integration**: OpenRouter (Claude) & Google Gemini.
- **Icons & UI**: Lucide React, Framer Motion (Animations), Sonner (Toasts).

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/abdulhaseebimran/Resume.IO.git
cd Resume.IO
```

### 2. Install dependencies
```bash
npm install
```

### 3. Setup Environment Variables
Create a `.env.local` file in the root directory and add the following:

```env
# Database
MONGODB_URI=your_mongodb_connection_string

# Authentication
NEXTAUTH_SECRET=your_nextauth_secret
NEXTAUTH_URL=http://localhost:3000

# AI Models
OPENROUTER_API_KEY=your_key
GEMINI_API_KEY=your_key

# Google OAuth
GOOGLE_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```

### 4. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see the magic!
